PUT /api/v2/user/<id>/premium/¶

Method for set premium and period premium.

Example:

Request:

URL: PUT http://127.0.0.1:8001/api/v2/user/3/premium/

{
    "is_premium": true,
    "premium_period_start": "2021-08-19T04:09:25Z",
    "premium_period_end": "2022-08-19T04:09:25Z",
    "plan_title": "Premium subscription",
    "plan_price": 19.99,
    "plan_description": "Premium"
}

Response:

{
    "id": 3,
    "is_premium": true,
    "premium_period_start": "2021-08-19T04:09:25Z",
    "premium_period_end": "2022-08-19T04:09:25Z"
}

Note

Status - 200

POST /api/v2/user/login/¶

Method for login as user. Authentication token not need.

Required fields:¶

  • email

  • password

  • device

Example:

Request:

URL: POST http://127.0.0.1:8001/api/v2/user/login/

{
    "email": "admin@gmail.com",
    "password": "qwerty",
    "device": "123456"
}

Response:

{
    "user": {
        "id": 69,
        "first_name": "First",
        "last_name": "Last",

POST /api/v2/user/change_password/¶

Method for change user password

Example:

Request:

URL: POST http://127.0.0.1:8001/api/v2/user/change_password/

{
    "current_password": "password",
    "new_password": "password123"
}

Response:

{
    "detail": "Your password has been changed successfully!"
}

Note

Status - 200

POST /api/v2/user/reset_password/¶

Method for send verification code for reset user password

URL: POST http://127.0.0.1:8001/api/v2/user/reset_password/

{

}

Response:

{
    "detail": "Verification code has been sent to your email!"
}

Note

Status - 200

POST /api/v2/user/reset_password_done/¶

Method for reset user password

{

    "code": "123456",
    "new_password": "password123456"
}

{
    "detail": "Your password has been changed successfully!"
}
