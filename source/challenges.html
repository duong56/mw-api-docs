Challenge API documentation (/api/v1/challenge/..)¶

POST /api/v1/challenge/¶

Method for create new challenge.

Required fields:¶

  • title

Example:

Request:

URL: POST http://127.0.0.1:8001/api/v1/challenge/

{
  "title": "new challenge"
}

Response:

{
  "id": 2,
  "title": "new challenge",
  "description": null,
  "deadline": "2020-11-15T19:18:46Z",
  "conditions": [],
  "image": null
}

Note

Status - 201 Created.
