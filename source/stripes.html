POST /api/v1/stripe/ephemeral_key/¶

Method for retrieve ephemeral key from stripe.

Required fields:¶

  • stripe_version

Example:

Request:

URL: POST http://127.0.0.1:8001/api/v1/stripe/ephemeral_key/

GET /api/v1/stripe/subscription/¶

Method for retrieve all subscriptions from stripe (limit 100).

Optional filter by status as GET parameter. One of: incomplete, incomplete_expired, trialing, active, past_due, unpaid, canceled, or all. By default value: all

Example:

Request:

URL:

GET http://127.0.0.1:8001/api/v1/stripe/subscription/?status=active

POST /api/v1/stripe/subscription/¶

Method for create stripe subscription.

Required fields:¶

  • item_price

  • default_payment_method

Example:

Request:

URL: POST http://127.0.0.1:8001/api/v1/stripe/subscription/
